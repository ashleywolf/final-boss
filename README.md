# Final Boss

Enter a GitHub username. Become a boss with a threat ranking, language-based attacks, and a card you can share.

**[Play it](https://ashleywolf.github.io/final-boss/)**

Your repos, stars, languages, and commit history get fed into a ranking system. C-rank newcomers get basic green cards. SSS-rank legends get rainbow gradients and anime boss sprites. Your primary language determines your class -- JavaScript makes you an Archmage, Rust makes you a Berserker, Python makes you a Necromancer.

Synthesized boss music plays. CRT scan lines flicker. Save the card and flex.

## How it works

- GitHub API fetches public profile, repos, stars, and languages
- 6-tier ranking system: C / B / A / S / SS / SSS
- 20+ language-to-class mappings for boss titles
- Pixelated boss sprite with neon glow and CRT effects
- Web Audio API synthesized soundtrack
- Canvas rendering with shareable card export
- Single HTML file, pure browser
